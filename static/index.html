Silicon Valley Insults API

About

Silicon Valley Insults API is a public RESTful API that delivers up humorous tech-themed insults from the television show "Silicon Valley" to multiple endpoints, where users are able to request content by season, episode, and character.

  • No API key needed.
  • Some insults contain strong language.
  • Default rate limit is 100 requests per hour.
  • In-depth documentation can be found below.
  • Enjoy!

Sample Request and Response

Sample request: 

const response = await fetch("https://silicon-valley-insults-0ff0d7448e81.herokuapp.com/api/insults");
const insult = await response.json();
console.log(insult);

Sample response: 

{
    season: 1,
    episode: 2,
    character: "Gavin",
    insult: "I hate Richard Hendricks, that little Pied Piper prick.",
}

API Endpoints

Get a random insult

Method
GET
Path
/api/insults

Sample request: 

const response = await fetch("https://silicon-valley-insults-0ff0d7448e81.herokuapp.com/api/insults");
const insult = await response.json();
console.log(insult);

Response format: 

{
    season: ...,
    episode: ...,
    character: "...",
    insult: "..."
}

Get a random insult by character

Method
GET
Path
/api/insults/character/{character}
Parameters
{character} is a string representing the first name of character you want an insult from

Sample request:

Errors and Exceptions

Insult does not exist

The insult you request may not exist. For example, you may request an insult from Jared from Season 1 Episode 3 when no such insult exists in the database. In this case, you will receive a 404 error with the following response:

Sample request: 

const response = await fetch("https://silicon-valley-insults-0ff0d7448e81.herokuapp.com/api/insults/season/1/episode/3/character/jared");
const insult = await response.json();
console.log(insult);

Sample response: 

{
"detail": "No insults available for season 1, episode 3, and character Jared"
}
